## コミットメッセージは書き始めの動詞をうまく選ぶ
　コミットメッセージは、「主語を省略し、動詞から書き始める」のが一般的  
　そのため、書き始めの動詞をうまく選ぶ必要がある  
　下記初めによく使われる動詞は、以下がある

- add      ：追加する
- allow    ：～を可能にする、許可する
- change   ：変更する
- clean    ：整理する(clean up)
- don't    ：～しないようにする
- fix      ：修正する
- implement：実装する
- improve  ：改善する
- make     ：～にする
- move     ：移動する
- refactor ：リファクタリングする
- remove   ：削除する
- rename   ：名前を変更する
- update   ：更新する
- use      ：使用する

　プロジェクトでどのような書き方がされているかを確認する
